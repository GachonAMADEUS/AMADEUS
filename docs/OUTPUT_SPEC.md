# Output Specification

This document defines the expected intermediate and final outputs of the AMADEUS pipeline.

## Input

```text
data/
  samples/
    foot_video.mp4
```

Required input:

- Foot video
- Checkerboard visible in the same scene

Optional input:

- Pre-extracted frames
- Precomputed COLMAP sparse model
- Precomputed 2DGS PLY

## Intermediate Outputs

### Frames

```text
outputs/frames/
  frame_0000.jpg
  frame_0001.jpg
```

Description:

- Representative frames extracted from the input video
- Blur, overexposure, and motion quality filtering applied

### Segmentation

```text
outputs/segmentation/
  foot/
  checkerboard/
  both/
```

Description:

- YOLOv11n-seg and SAM-refined segmentation results
- Separate foot-only, checkerboard-only, and combined-mask images

### COLMAP

```text
outputs/colmap/
  database.db
  sparse/0/
    cameras.bin
    images.bin
    points3D.bin
```

Description:

- Camera poses and sparse point cloud
- Initial input for 2DGS training

### 2DGS

```text
outputs/2dgs/
  reconstruction.ply
  mesh.ply
```

Description:

- 2D Gaussian Splatting training result
- Extracted mesh result

### Scale Debug

```text
outputs/scale/
  projected_checkerboard.png
  debug_lines.png
  scale_factor.json
```

Description:

- Checkerboard-based scale factor calculation outputs
- Debug images and computed scale values

## Final Outputs

```text
outputs/final/
  final_foot.stl
  final_foot_scaled.stl
  measurements.json
  report.json
  report.txt
  sliced_project.3mf
```

### `final_foot.stl`

The postprocessed foot mesh.

Expected properties:

- Foot-only geometry
- Major floating artifacts removed
- Repair/simplification applied where needed

### `final_foot_scaled.stl`

The STL after real-world scale correction.

Expected properties:

- Checkerboard-based scale factor applied
- Loadable in Bambu Studio or OrcaSlicer

### `measurements.json`

Foot measurements and mesh metadata.

Example schema:

```json
{
  "scale_factor": 1.0,
  "foot_length_mm": 0.0,
  "foot_width_mm": 0.0,
  "instep_height_mm": 0.0,
  "bounding_box_mm": [0.0, 0.0, 0.0],
  "triangle_count": 0,
  "watertight": false
}
```

### `sliced_project.3mf`

The project file generated by Bambu Studio or OrcaSlicer.

Notes:

- Bambu Cloud upload is a separate future step.
- Printer/profile settings may differ by execution environment.
